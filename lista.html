Questionário

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    2. O atributo placeholder especifica uma dica curta que descreve o valor esperado de um campo de entrada.

  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. Serve para o <input> com atributo type="radio" ou type="checkbox" vim selecionado sem necessidade do usuario selecionar.

  3. Na tag <input> para que serve o atributo "value"?

    4. O atributo value é um valor predeterminado que não desaparece após interagir com o input.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. O atributo name.

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. A tag <label> define um rótulo para elementos do formulário. O atributo for serve para interligar o elemento relacionado pela tag label.

  6. Na tag <option> para que serve o atributo "selected"?

    7. Determina que uma opção venha pré-selecionada quando a página é carregada.

  7. Nas tags de formulário para que serve o atributo "required"?

    8. Ele determina que um campo de inserção deve ser preenchido antes de enviar o formulário.

  8. Para que serve a tag <textarea>?

    9. É um campo de entrada de múltiplas linhas.

  9. Na tag <form> para que serve o atributo "action"?

    10. Define para qual página os dados do formulário serão enviados.

  10. Na tag <form> para que serve o atributo "method"?

    11. O method nos permite selecionar de qual maneiras os dados serão passados quando submetemos um formulário, eles podem ser get ou post. No método get os parâmetros são passados no cabeçalho da requisição. Já no post ele envia os parâmetros no corpo da requisição.

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    12. É utilizado o método post mas não significa que os dados estão protegidos só por que não aparecem na URL.

  12. Qual valor deve ser informado no atributo "method" na tag
    em um formulário que será usado para fazer upload de arquivos?

    13. O método post é o melhor caminho já que ele encapsula os dados no corpo da requisição e consegue transportar mais dados.

  13. O que as tags <fieldset> e <legend> produzem?

    14. A tag <fieldset> serve para agrupar elementos dentro de um formulário e a tag <legend> cria uma legenda para o seu elemento fieldset.
